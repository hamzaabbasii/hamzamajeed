Real-Time Chat App

Secure messaging, image sharing, and offline support.

Project Overview

This project is a full-featured, real-time chat application developed using Flutter and Firebase. The primary goal was to create a seamless and secure messaging experience for users across both iOS and Android platforms.

Key Features

  • User Authentication: Secure email/password and Google Sign-In options.
  • Real-Time Messaging: Instant message delivery using Firebase Firestore.
  • Image Sharing: Users can send and receive images, stored in Firebase Storage.
  • Offline Support: Messages can be composed and read offline, syncing automatically.

Technology Stack

  • Flutter & Dart
  • Firebase Firestore
  • Firebase Auth
  • GetX State Management
