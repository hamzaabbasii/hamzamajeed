Real-Time Chat App

Secure messaging, image sharing, and offline support.

Project Overview

This project is a full-featured, real-time chat application developed using Flutter for the front-end and Firebase for the backend. The primary goal was to create a seamless and secure messaging experience for users across both iOS and Android platforms. The app allows for instant text messaging, image sharing, and ensures that messages are delivered even with intermittent network connectivity through Firebase's offline persistence.

Key Features

  • User Authentication: Secure email/password and Google Sign-In options.
  • Real-Time Messaging: Instant message delivery using Firebase Firestore.
  • Image Sharing: Users can send and receive images, stored in Firebase Storage.
  • Offline Support: Messages can be composed and read offline, syncing automatically when the connection is restored.

Technology Stack

  • Flutter & Dart
  • Firebase Firestore
  • Firebase Authentication
  • Firebase Storage
  • GetX (State Management)
